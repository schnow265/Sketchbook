# Security Policy

## Supported Versions

This Repo will *only* support current versions, backporting is not planed. Also, not porting to other Modloaders.

| Version | Supported          |
| ------- | ------------------ |
| newest Version   | :white_check_mark: |
| lower than 1.19.4   | :x:                |
